# Imagens

Pasta destinada a sprites, fundos, ícones e outros elementos visuais do jogo.

## Conteúdo Atual

- `fundo.png`: Fundo principal do jogo.
- `personagens/`: Pasta contendo as sprites do personagem.

## Recomendações

- Separe imagens por tema quando o projeto crescer.
- Mantenha dimensoes e padroes consistentes para facilitar colisao e animacao.
- Prefira formatos com transparencia quando necessario (ex.: `.png`).
